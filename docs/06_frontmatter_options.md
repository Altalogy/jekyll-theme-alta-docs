## Frontmatter options

Some of the Frontmatter properties are displayed in the navigation and in the document header.

```md
title: My title
category: Features
layout: doc     # other options: page
order: 501      # used to sort documents in the navigation
toc: true       # display Table of content

vtag: my tag           # Tag - displayed in the nav and doc header
vtags: [tag 1, tag2]   # List of tags - displayed in the nav and doc header
status: pause          # Status tag - displayed in the nav and doc header
prio: high             # Priority tag - displayed in the nav and doc header
system_components: [admin]  # List of system components tag - displayed in the nav and doc header
version: 1.0           # Version - displayed in the nav and doc header
```
